## Want to Run It Locally?

1. **Clone the repo**

   ```bash
   git clone https://github.com/Brown-Sage/Portfolio_Website
   cd Portfolio
   ```

2. **Install dependencies**

   ```bash
   npm install
   ```

3. **Start the dev server**

   ```bash
   npm run dev
   ```

4. **Open your browser** to `http://localhost:5173/`
